中断门

0x01中断门

中段门Type中的D为1，表示这是一个32位的中断门，为0则是一个16位的中断门，中断门存储在idt表中。idt表中包含任务门描述符、中断门描述符、陷阱门描述符。

中断门描述符与调用门一样，访问中断门需要cpl=dpl，根据段描述符中的段选择子来决定是否提权，然后根据段选择子会查找gdt表。

调用中断门通过 int n来调用，n代表idt表的下标，然后进入中断门后通过iret或者iretd返回,中断门会压栈（提权）ss，esp，eflags，cs，eip，不提权cs，eflags、eip

0x02中断门实验

#include<stdio.h>
#include<stdlib.h>
//地址411740
void __declspec(naked) HighLevel()
{
    _asm {
        int 3;
        iretd;
    }
}

int main()
{
    
    __asm {
        push fs;    //与调用门同理
        int 0x20;
        pop fs;
    }
    system("pause");
}

0x03陷阱门

陷阱门与中断门并无二致，只不过陷阱门在idt表中
